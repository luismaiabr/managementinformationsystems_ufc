# managementinformationsystems_ufc

## Documentação do Sistema (SIG)

Bem-vindo à documentação do projeto. Esta documentação serve como base e define os requisitos para a construção de painéis gerenciais e dashboards em ferramentas de Business Intelligence (BI), como Power BI, Tableau, Metabase, entre outras.

O objetivo principal destes documentos é orientar o desenvolvedor na implementação técnica das métricas de negócio. O programador deve utilizar o mapeamento e o dicionário de dados aqui presentes para extrair e implementar exatamente os indicadores (KPIs) e resultados-chave (key-results) que foram definidos no planejamento.
