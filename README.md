# managementinformationsystems_ufc

## Documentação do Sistema (SIG)

Bem-vindo à documentação do projeto. Esta documentação tem como objetivo principal orientar o desenvolvedor na construção de painéis gerenciais (dashboards) utilizando ferramentas de Business Intelligence (BI) como Power BI, Tableau ou Metabase.

### 1. Documento de Requisitos (Dashboard)

O documento a seguir é o guia principal. Ele define exatamente quais indicadores funcionais (KPIs), resultados-chave (key-results) e filtros devem ser implementados na dashboard final. Toda a implementação visual deve basear-se nele.

- [Requisitos para Dashboard (BI)](./docs/04_requisitos_dashboard.md)

### 2. Documentos de Apoio

Estes documentos servem como base técnica e de negócio. Eles contêm o detalhamento dos processos operacionais e o dicionário de dados que embasam os requisitos solicitados acima.

- [Processo 1: Suporte às Universidades](./docs/01_processo_suporte_universidades.md) - *Detalha as regras e fluxos de atendimento.*
- [Processo 2: Ciclo de Vida do Contrato de Estágio](./docs/02_processo_ciclo_vida_contrato.md) - *Mapeia as etapas e SLAs de assinaturas.*
- [Detalhes Técnicos: Dicionário de Dados e Mapeamento](./docs/03_detalhes_tecnicos.md) - *Guia técnico conectando os processos às estruturas do banco de dados.*
